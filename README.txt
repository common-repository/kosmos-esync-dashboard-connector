Connect point of sale, ERP and ecommerce applications. Sync inventory, transfer orders and manage product data in one place with the eSync Dashboard.

== Description ==

Kosmos eSync Dashboard Connector allows you to access the Kosmos eSync Dashboard from inside WordPress. Existing eSync users can run Tasks to sync inventory and orders between applications and manage integration settings.

Kosmos eSync integrates data between leading point of sale, ERP, marketplace and ecommerce applications.

Current integrations include:

* WooCommerce* Revel Systems Point of Sale* Lightspeed Retail Point of Sale* Vend Point of Sale* Acumatica ERP* Integrate Amazon and eBay using marketplace tools* Clover Point of Sale
* View a complete list of integrations [here](https://www.kosmoscentral.com/connections).

Features of Kosmos eSync Dashboard Connector:

* Quick link to Kosmos eSync Dashboard is added to the top menu of your WordPress admin area
* Quick link to help documentation
* Access to video tutorials
* Access to support center
* Requires the WooCommerce plugin to be installed

See our terms of use at [kosmoscentral.com](https://www.kosmoscentral.com/terms-and-conditions).

== Installation ==

1. Upload `ked-connector.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. The sidebar will now have a link to access the Kosmos eSync Dashboard from your WordPress admin area
